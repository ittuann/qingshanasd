在情绪、感官、社交、学习、工作等方面，孤独症人士面临着多重困难。


**这些困难不仅源于自身的差异和障碍，更来自外界的不理解和社会性支持的缺乏。**

而这些困难，往往未被合理呈现。


正如青衫所述：

> 很多时候我感到绝望但意识不到自己需要帮助，或者是意识到自己需要帮助但是不知道需要怎么样的帮助。
> 
> **困境需要被看见和描述。**

  

**在浪漫化和污名化之外，是真实的人和真实的生活。**

  

因此，青衫向群友们征集了一些当事人视角的体验与感受。

  

> **目前，系统层面的主要问题包括：**
> +   社会对不同能力和表现的包容度低
> +   大众对孤独症的的误解和偏见严重，对孤独症人士的需求缺乏理解和接纳
> +   **教育、医疗、社会支持等多个体系，都缺少孤独症意识，急需系统性改进**

 ### **01 情绪与心理状态**

  

**社会对孤独症人士的情绪差异及需求缺乏理解和接纳，孤独症人士常被误解或被忽视。**

  

同时，由于缺乏有效的情绪管理策略与支持系统，孤独症人士难以得到有效帮助。

  

这可能导致孤独症人士：

  

+   持续的情绪崩溃和焦虑状态，自我否定和心理压力巨大
+   因为无法适应而产生负面经历，从而缺少或失去机会，影响社会融入

  
---
**（1.1）个人体验**

**以下摘录总结自群友分享**

> +   感觉自己的“情绪情感”和“理智思维”是分离的，难以识别和判断自己的情绪（困困）
> +   记忆过于清晰，思维过于活跃，无法忘记痛苦回忆，易被触发（困困）
> +   被激烈的内在情绪和思维耗尽，没有任何精力做自己喜欢擅长的事，感觉被困在了自己的脑子里（困困和朋友们）
> +   对理解和表达抽象的事物有困难，比如情绪（冰糖猫猫）
> +   察觉不到别人的情绪，但对自己的负面情绪敏感（芍芍）
> +   经常因为“较真”陷入暴怒的情绪（逆转 1001 次拯救金毛小狗）
> +   没有情感记忆，只能以旁观的视角描述当时发生了什么（小藻）
> +   难以察觉自身情绪，很多时候靠身体疼痛来察觉情绪（小藻）
> +   情绪延迟，无法及时感受到应有的情绪，之后情绪才来（伊具施缇）
> +   意识不到或不会描述自己遇到的问题，不知道怎样评估自己的状态是否需要寻求帮助，不知道如何寻求帮助（zzz、困困、青天外）
> +   经常只能一个人坚持最后情绪崩溃，被其他人指责脆弱，但自己都不知道发生了什么（困困）
> +   精神系统周期性崩溃（散人）

---

**（1.2）外在反馈**

**以下摘录总结自群友分享**

> +   有激烈情绪但表情平静，于是被指责“冷漠无情”（困困）
> +   要么觉得我们不需要帮助，要么批评“连吃饭都做不到，废物”，不肯帮助理解，甚至从不听我说我需要什么帮助（困困和朋友们）

---

**（1.3）结果**

**以下摘录总结自群友分享**
  
> +   初中因为抑郁症、焦虑症休学住院，高中无法适应学校生活再次休学，成年后融入社会困难（蒙布朗）
> +   怀疑自己（散人）

  

 ### **02 感官与身体反应**

  

孤独症人士在感官和身体反应方面与神经典型群体存在差异，在生活中面临巨大挑战，需要环境和行为上的适应与支持，却常被误解为矫情、任性、懒惰或不配合。

  

**而社会理解和支持的缺乏，加重了孤独症人士日常生活中的困扰和不适。**

  

这可能导致孤独症人士：

  

+   生活质量受到严重影响，身心状态得不到有效保障
+   因他人的不理解不接纳而社交困难，被孤立和排挤
+   自我否定和怀疑

---

**（2.1）个人体验**

**以下摘录总结自群友分享**


> +   对食物的口感和味道敏感，导致几乎没有什么可以吃的东西，甚至吞东西都吞不下（困困）
>     
> +   共现中枢听觉处理障碍，听不清别人说话（将七、星星与白桦）
>     
> +   内感受障碍，对自己身体内部的感受非常迟钝，不知道渴了饿了热了冷了是什么感觉，需要人帮助提醒，会经常低血糖/感冒等（Andreth、困困）
>     
> +   有各种各样的重复动作，比如转圈圈、绞手、咬指甲、重复短词短句子到我妈生气叫我闭嘴，经常克制自己，怕别人觉得我奇怪（冰糖猫猫）
>     
> +   肢体不协调，学动作慢或学不会，走路姿势也被说过有点不一样（冰糖猫猫）
>     
> +   手脚不协调，本体感觉差，经常摔跤或者撞到东西（Andreth、琉璃）
>     
> +   运动能力远远落后（蒙布朗）
>     
> +   感官超敏，易过载，所有的声音都会被放大传到耳边，会因此难受（琉璃、依依、冰糖猫猫）
>     
> +   味觉和触觉超敏：挑食，被描述为“只吃固定的食物”；从小只能用一种类型的床单被套，不然睡不着（冰糖猫猫）
>     
> +   触觉敏感，不喜欢衣服标签、长袖长裤、手套手表等接触皮肤（琉璃、冰糖猫猫、伊具施缇）
>     
> +   无法感知自己形体的存在，需要外部施加物理重力，否则无法安静坐着，会因为“感受不到自我”而本能地进入警觉不安的恐惧之中，无法控制（困困）
>     
> +   长期失眠，睡不着，不知道为什么（兔狲）

---

**（2.2）外在反馈**

**以下摘录总结自群友分享**

> +   被认为是矫情、敏感、任性（真流慎吾）


---

**（2.3）结果**

**以下摘录总结自群友分享**
  

> +   运动能力发展慢于同龄人，体测经常倒数（蒙布朗）
>     
> +   因为较差的运动能力，慢慢变宅；又因为宅、不运动，体质变差（蒙布朗）
>     
> +   肢体不协调，因为学得特别慢，被同学放弃，直接让我不参与活动（冰糖猫猫）

  

 ### **03 社交与沟通**

  

**社会对孤独症人士的社交困难有严重误解和偏见。同时，孤独症人士缺乏有效的社交支持和指导。**

  

这可能导致孤独症人士：

  

+   难以建立和维持良好社交关系，缺乏支持网络，难以融入群体
    
+   被误解、孤立、排斥、霸凌
    
+   持续的情绪困境和孤独感
    
+   过度伪装带来更多的内心压力和疲惫

---

**（3.1）个人体验**

**以下摘录总结自群友分享**

  

> +   无法理解和意识到社交中的规则（地球仪）
>     
> +   无法解读话语中隐藏的信息，例如情绪、态度、话里有话等（芍芍）
>     
> +   不懂暗示，不会说话“好听”（冰糖猫猫）
>     
> +   有强烈社交需求，但根本不知道如何社交；没有朋友，总是孤单和不知所措；加入不了对话，感觉格格不入（冰糖猫猫）
>     
> +   被曲解意图但由于表达能力有限而无法解释清楚（将七）
>     
> +   不懂得如何跟朋友聊天、相处，经常跟同学发生矛盾（琉璃）
>     
> +   不知道怎么关心喜欢的人或与之相处（小藻、伊具施缇）
>     
> +   容易沉浸在自己感觉里完全忽视掉身边的人（小藻）
>     
> +   难以预测他人的行为和反应，与人合作困难（Kenopsia）
>     
> +   难以融入集体并适应环境，为了应付工作中社交而不断内耗自己（依依）
>     
> +   很难建立非依恋的社交关系（匿名）
>     
> +   想建立良性关系，但很多时候不知道怎么做（潇妍喵）
>     
> +   因为社交受挫而回避社交，没有朋友（匿名）
>     
> +   为了融入群体和自我保护而过度 masking，很难在其它社交情境中摘下面具，因此常常被人误解，导致内耗严重（永远 5 岁的猫铃薯）
>     
> +   因为不明白别人的动机，总会不小心冤枉好人，相信坏人，被利用（树仔）
>     
> +   过于诚实，时常过度分享，难以处理朋友以下的熟人关系（双 A 宅宅）
>     
> +   道德感和尽责性过高，时常被压榨劳力（双 A 宅宅）
>     
> +   和家人之间误解不断，在重要问题上难以有效沟通（落英）
>     
> +   面对面交流时，脑子转不动，又因为不安焦虑，反应慢；因为对模糊话语的理解不同，或者对方无视精确性限定，会出现沟通误差；由于他人无法理解此类困难，情绪压力增高（匿名）
>     
> +   想要结交现实中的孤独症朋友，希望有互助组织（宁静者）
>     
> +   职场不知道该说什么话题，只能被动交流，自己找的话题别人都觉得奇怪（兔狲）
>     
> +   社交的时候总是短路，感觉眼睛、耳朵、嘴和脑子最多只能同时用两样（喋喋鹦鹦）
>     
> +   事后才意识到自己好像在某个场合当中表现奇怪了，但想不通也没法描述具体是哪里奇怪，在持续的困惑和混乱中精疲力尽，内耗好久才能缓过来，缓过来之后又不得不再去社交（喋喋鹦鹦）
>     
> +   不知道怎么和人找话题，即使有人搭话也要想很多才能回复，总是游离在别人的讨论之外，但是长时间没人说话又会特别孤独，同时有社交焦虑（blbdtz）
>     
> +   眼神接触非常困难，因为会因此紧张焦虑、无法继续思考我在说的内容（冰糖猫猫）
>     
> +   可以出去社交，但是会突然 burn out，恢复期很长，不可控（71）

---

**（3.2）外在反馈**

**以下摘录总结自群友分享**

  

> +   被认为内向、懒（Stamp）
>     
> +   容易被扣上“冷漠、没礼貌、不好相处”的帽子（匿名）
>     
> +   无法做到的事被他人认为“不想做”，自救行为被认为是“自甘堕落”（将七）
>     
> +   听不懂言外之意就被认为是幼稚、不够成熟（真流慎吾）
>     
> +   社交中，别人让我感到不适，我会被指责为“过于敏感，胡思乱想”；我让别人感到不适，会被指责为“故意使坏，成心作恶”（苯苯）
>     
> +   周边人不理解，经常被责备“哪有这些毛病，少看些网上的东西”（依依）
>     
> +   被要求“像一个成年人”一样解决与他人社交的问题（真流慎吾）
>     
> +   因为看不懂别人的暗示，就被家人说不懂事、听不懂是开玩笑、说话太直、不会说话（冰糖猫猫）

---

**（3.3）结果**

**以下摘录总结自群友分享**

  

> +   常常被人误解，内耗严重（永远 5 岁的猫铃薯）
>     
> +   担心以后找不到工作（冰糖猫猫）
>     
> +   因为社交困难，更容易被孤立、霸凌（冰糖猫猫）

  

 ### **04 学习工作与日常生活**

  

**孤独症人士需要更多个性化的学习方式和资源，在现有教育系统中难以获得有效支持和理解。**

  

这可能导致孤独症人士：

  

+   无法有效学习，未能发挥自己的潜力，学业和职业发展受限
    
+   持续的焦虑和压力，自我否定感强烈
    
+   被误解和批评


---

**（4.1）个人体验**

**以下摘录总结自群友分享**

  

> +   脑子单线程，只能高速单独处理听觉或者视觉，常规课堂像地狱（困困）
>     
> +   需要更多的反应时间，脑子没法同时处理几个东西，好几次都被朋友家人说，我会难过，因为我真的做不到（冰糖猫猫）
>     
> +   只能单线程处理事情，影响效率（芍芍）
>     
> +   被迫以自己无法接受的形式学习，而不被允许用自己的方式快速掌握和理解（困困）
>     
> +   不明白很多事物的关联，不知道怎么做有效笔记，听课感觉没有重点（Aira）
>     
> +   有时候会进入过度专注的状态，无法转移注意力去做更要紧的事，感觉动弹不得（Andreth）
>     
> +   难以自行切换做事的模式，明明已经快累崩溃了但是无法停止，只能累到自己彻底瘫痪（困困和弟弟）
>     
> +   严重走神，无法集中注意力，导致工作经常出错（小藻）
>     
> +   因为教室环境有很多人坐在一起而紧张（伊具施缇）
>     
> +   学校环境下学习状态差，持续陷入焦虑状态（江离）
>     
> +   合并 ADHD 外加手部重复动作，执行力严重受损（依依）

---

**（4.2）外在反馈**

**以下摘录总结自群友分享**

  

+   被认为没有智力发育障碍就不需要帮助（困困）
    
+   被指责为“懒”或“矫情”，得不到锻炼独立的机会（困困）
    
+   用听力学习而眼神没看黑板，被老师批评“走神”（困困）
    
+   被认为“智力高”但与糟糕学业成绩不符（江离）
    
+   被父亲指责“你做不到洗澡吃饭出门只是因为意志力还不够”，而当我努力克服了之后，对方只是觉得“大家都会，你本来就该会，没什么大不了的”（困困）
    
+   给人一直在动的印象（依依）
    
+   很多人只会觉得是“能坚持”“倔强”，不理会我的“我停止不了，请打断我让我停下来”（困困和弟弟）


---

**（4.3）结果**

**以下摘录总结自群友分享**

  

+   被迫学了不喜欢的东西以后一点也学不进去，结果休学留级了，非常困扰（伊具施缇）
    
+   总是无力绝望地看着自己被耗尽而其他人乐在其中（困困和弟弟）
    



 ### **05 医疗**

  

**孤独症人士在现有医院环境中感到不安。**

  

这可能导致孤独症人士：

  

+   拖延就医，不知道如何及时且有效地寻求帮助
    
+   身心健康得不到重视和保障，影响日常生活和工作


---

**（5.1）个人体验**

**以下摘录总结自群友分享**

  

> +   因为害怕医生态度、医院环境、疼痛体验、就诊描述不清等，而害怕去医院，并因此拖延就医（将七）
>     
> +   感觉身体一直不太好，不知道要怎么检查或者改善（潇妍喵）
>     

  

 ### **06 自我认同与认知**

  

**社会对孤独症人士缺乏接纳，孤独症人士的自我意识和需求常得不到基本的认可和尊重。**

  

这可能导致孤独症人士：

  

+   在现有社会环境中常感到自己是“异类”，并为了融入社会而过度伪装
    
+   面临自我认同困境，自我认同感低，自尊心受损，感到压抑和自我否定


---

**（6.1）个人体验**

**以下摘录总结自群友分享**

  

> +   很难找到自我认同感，觉得自己是个怪胎，或者异类（Clever woman）
>     
> +   为了融入和成就而伪装自己，找不到真实的自己，自己忍受压力和抑郁（匿名）
>     
> +   不被尊重和重视，反被指责不应该不适，并被要求强行适应和改正（热心市民 Vince）
>     
> +   冒名顶替综合症，不敢接受帮助（Andreth）
>     
> +   就算遇到了愿意帮助我的人也有不配得感，不想给别人添麻烦（树仔）
>     
> +   极大的自我否定（依依）
>     
> +   在日常生活中，自我的意识并没有得到基本的认可（散人）
