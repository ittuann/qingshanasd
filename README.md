## 🛠️ Develop

1. Clone the Repository

```bash
git clone https://github.com/ittuann/qingshanasd.git
cd qingshanasd
```

2. Install Dependencies

```bash
pnpm i
```

3. Preview

```bash
pnpm dev
```

To preview the builded static site:

```bash
pnpm build
pnpm serve
```

Then you can open `http://localhost:3000` in your browser.
